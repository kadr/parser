## Parser
Запуск 
```
make run
```

В Makefile - е заданы все переменные, достаточно просто запустить

CATEGORY_NAME - название нужной категории

LIMIT - количество доставаемых данных, опционально, по умолчанию 100

PROXY - адрес прокси в формате https://ip-address:port, опционально, если пустой или не задана переменная, будет без него работать.

LENTA_COOKIE - куки нужные для корректного прохождения запроса на сайте ленты

Что не успел:

1. Разобраться с куками. Для того что бы запрос на получение товаров с сайта ленты проходил нормально в заголовках нужно передавать поле Cookie, его нужно видимо как то отдельным запросом получать, не успел разобраться, поэтому вынес это в перемнные окружения в Makefile

2. Нужно добавить переменную для записи в нее название маркета, который нужно парсить, в main файле читаем его, сплитуем и запускаем столько горутин, сколько маркетов было задано, пишем результаты в канал и потом собираем все данные, со всех маркетов.
